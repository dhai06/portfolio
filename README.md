# Portfolio
> *A Hinge-inspired personal website built with React and Next.js.*

This portfolio is designed to mimic a premium mobile app experience, featuring a card-based interface, smooth Framer Motion animations, and a unique "profile" browsing UX.

## Features
*   **Clean UI**: A clean, card-based interface inspired by modern dating apps.
*   **Interactive Elements**: Draggable rails, heart button interactions, and responsive modals.
*   **Animations**: Fluid transitions and gestures powered by Framer Motion.
*   **Responsive Design**: Fully optimized for both mobile and desktop experiences.

## Tech Stack
*   **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/)
*   **Icons**: [Lucide React](https://lucide.dev/)

## Project Structure
*   `src/components`: UI components (Blocks, Modals, Rails).
*   `src/data`: Portfolio content and data structure (`portfolioData.ts`).
*   `public`: Static assets and images.
